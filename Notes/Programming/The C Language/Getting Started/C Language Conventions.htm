Strings

Strings are a collection of characters (and symbols) that can be used in a program.

To create a string, enclose a your collection in either single or double quotations

ie ' or

 

 

Comments

A comment is text that is not interpreted by the compiler, and has no effect in the execution of the code.
In other words, comments exist solely to help programmers quickly explain to others what is going on.

Comments should include a precise explanation of the execution of the statement

To create a comment, prepend a "//" to your code

You can also comment a group of lines with a multiline comment block "/* … */"

Example

// This entire line is a comment printf("Hello, world!");

printf("Goodbye, world!"); // This line will execute because the comment is after the statement.

Output

Goodbye, world!

 

 

Libraries

A library is essentially code that someone else has already written. We can use these libraries in our own programs to save time.

No point reinventing the wheel, right?

Example

#include <stdio.h> // This line loads the code of stdio.h into the program

 

 

Semi-colons

Semi-colons (";") are required at the end of every statement to separate that statement from the next.

 

 

Functions

A function is a block of code that can be reused again and again.

Some special function names exist, such as the main function, which executes automatically when the program is run

In the C language, we must explicitly state the type of a function's result.

 

For all return types except for void, the return statement, followed by a result is required

 

Syntax

returnType functionName (arguments) {

  ... // Do stuff

  return RESULT;

}
