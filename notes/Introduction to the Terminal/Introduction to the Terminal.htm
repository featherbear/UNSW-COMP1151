What is a terminal?

In programming terms, a terminal is an interface that allows access to the functions of the system

 

Before the days of graphical desktop interface, computer users would type out commands in a terminal to instruct the computer on what to do.

 

Nowadays, graphical desktop interfaces like Windows and Mac OS X have replaced the need for the terminal in favour of a more user-friendly interface.

 

Why use a terminal?

  1. It's faster

How do you make a folder on Windows? right-click, new folder, type in name, press enter
How do you make a folder in a terminal? mkdir name, press enter
... see. Much faster.

 

  1. You can't do everything on a graphical interface

The graphical interface is often limited in the things it can do. As a programmer we want to utilise everything that the computer can offer, so thus we gain access to these hidden features in a terminal.

 

  1. We're cool people

Why use a graphical user interface?

  1. Easier to do image-based work, such as editing an image
