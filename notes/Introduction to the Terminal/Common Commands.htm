Common Commands

Tuesday, 27 February 2018

2:26 PM

$> cd DIRECTORY

changes directory to the directory called DIRECTORY

 

$> cd ..

Changes the directory to the parent folder

 

$> mkdir DIRECTORY

makes new directory called DIRECTORY

 

$> rm FILE
removes a file called FILE
(will not work for a directory)


$> cat FILE

Show the contents of FILE
 

$> gedit FILE

Opens the file FILE the graphical text editor. If no filename is supplied an empty document is created

 

Cool things!
- When using the terminal, you can autocomplete file and directory names with the  key

- if you need to run a program in the background, append a space and an ampersand & at the end of the line
  ie "gedit &"

- In a terminal you can retype the previous command(s) by pressing the UP arrow key
