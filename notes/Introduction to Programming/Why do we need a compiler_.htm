Why do we need a compiler?

Wednesday, 28 February 2018

9:39 AM

Most languages are developed such that their code is easy to understand by humans.

However, computers require that code is written in a certain format. (This format is also dependent on the computer)

In essence, a compiler translates the source code into a binary file, which instructions that the computer can understand and run
