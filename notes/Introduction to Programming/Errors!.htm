Forgetting to close your strings

Cause: When writing code, strings need to be enclosed in a matching pair of quotation marks
Solution: Ensure that you start and end a string with a (single/double) quotation mark, and end the string with the same quotation mark

"Hello, World" - using a pair of double quotation marks - VALID

'Hello, World' - using a pair of single quotation marks - VALID

'Hello, World - using only one single quotation mark at the start - INVALID

Hello, World" - using only one single quotation mark at the end - INVALID

'Hello, world" - using a mixture of single and double quotation marks - INVALID
