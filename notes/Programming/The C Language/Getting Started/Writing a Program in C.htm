To run a program written in C, we need to first compile the source code into an executable program

 

Let's say that we have just logged onto a terminal, and our directory looks like this:

 

HOME

+---Desktop
+---Downloads

\---Example

    \---sourceCode.c

 

To make things easier, let's navigate into the directory called Example

$> cd Example

 

Now we can run the compiler

$> dcc -o outputProgram sourceCode.c

This command runs the program dcc, compiles the contents of sourceCode.c into a program called outputProgram

 

Once the compiler has finished, we will be able to run the program

$> ./outputProgram

If sourceCode.c was the hello world example, then we should see Hello, world! being displayed
