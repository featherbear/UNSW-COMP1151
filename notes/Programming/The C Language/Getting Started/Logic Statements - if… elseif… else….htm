Logic Statements - if… elseif… else…

Wednesday, 28 February 2018

3:26 PM

Often a program needs to perform different actions depending on different cases. To achieve this, we use logic statements in the program.

 

For example, checking whether someone is (legally) able to drive.

Logically, the program needs to check the following criteria:

  • The person's age must be 16 years above to be allowed to drive
  • If the person is under age (15 years or younger) they are not allowed to drive

 

 

Jumping into C

Syntax

The basic syntax for these conditional statements are as follows

if (conditionOne) {

... do stuff ...

} elseif (conditionTwo) {

    ... do stuff ...

} else {

... do stuff ...

}

 

Explanation

A  is a statement that, if evaluated to as true, will execute the code within that  statement

We start off the conditional statement with the  keyword.

If we have more conditions to test, we use the  keyword.

We can run code that executes if none of the previous conditions were met with the  statement.

 

Multiple conditions

&&

||

 

Example

#include <stdio.h>

int main(void) {

int age;

printf("What is your age?\n");

scanf("%d", &age);

 

if (age >= 16) {

printf("You can drive!\n");

else {

printf("You are too young to drive!\n");

}

 

printf("Have a nice day!!!\n");

return 0;

}

 

Explanation

 

#include <stdio.h> // Loads the library that contains both the printf and scanf functions

int main(void) { // Start of the main code function…

int age;  // Define an integer variable called `age`

printf("What is your age?\n");  // Prints out the line What is your age?

scanf("%d", &age);  // Reads the user input as an integer into the variable `age`

 

if (age >= 16) {  // Check for the condition if `age` is greater than or equal to 16

// Stuff in here will execute ONLY if the age is greater than or equal to 16

printf("You can drive!\n");

else {

     // Stuff in here will execute if the condition is evaluated to be incorrect

printf("You are too young to drive!\n");

}

 

printf("Have a nice day!!!\n");

return 0; // Finish the main code function…

}  // Close the main code function
