Constants

We can use constants to define arbitrary values (strings, booleans and integers) that we probably don't want to remember, or do not need to be changed

#define MINIMUM_DRIVING_AGE 16;

Variables or constants?

You could use either, but constants make it look cleaner, plus you could accidentally change the value

 

What are variables

A variable is like a pocket, where you can store things in them!

 

Name your variables

Why? Though it is not crucial, it is recommended to name your variables meaningfully, so that other people (and you yourself) are able to easily understand what the variable does.

Rules

  • Must start with either an uppercase letter, lowercase letter, or an underscore
  • Can then include any amount cased letter, number, or underscore
  • Cannot be a keyword from C (compiler won't compile)

(regex) ^[a-zA-Z_][a-zA-Z0-9_]*$

Types of variables

The most common variables in C are as follows

str - String - Holds text and character. To make a string, enclose your text in quotation marks ie "HELLO"

int - Integer - Representation of integer numbers (ie 0, 1, 2, 500, -9)

float - Floating Point Numbers - These are decimal numbers (ie 1.555, 3.141592)

bool - Booleans - Either a true or false value

 

Defining a variable

To initialise a variable without a value

TYPE NAME;

For example

str school;

 

To initialise a variable with a value

TYPE NAME = VALUE;

For example

str school = "UNSW";
