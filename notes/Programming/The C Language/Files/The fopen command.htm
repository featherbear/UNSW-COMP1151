Overview

The  function is the function used to access a file.

In combination with  and , we have access to reading and writing values in a file

 

Declaration

Parameters

Remarks

If  could not be accessed,  will return , and sets a special variable .

If we catch when , we therefore have a method to detect errors.

These errors usually include when a file does not exist.

 

Example (Reading an existing file)

FILE *file = fopen("hello.txt", "r");

if (file == NULL) {

    perror("Error: ");  // prints why the open failed

                        // perror: "Error: " + value of errno

    return -2;

}

 

int c;

while ((c = fgetc(file)) != EOF) {

    fputc(c, stdout); //

                      // putchar(c) performs the equivalent operation

}

fclose(file); // Close the file

 

Example (Writing a new file)

FILE *file = fopen("hello.txt", "w");

if (file == NULL) {

    perror("Error: ");  // prints why the open failed

                        // perror: "Error: " + value of errno

    return -2;

}

 

int c;

fprint(file, "Hello %s", "world"); // Same as printf, but the first argument is a FILE

fclose(file); // Close the file
