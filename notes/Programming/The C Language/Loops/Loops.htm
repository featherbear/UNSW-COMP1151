Introduction

In programming, loops allow you to execute lines of code repetitively until a condition is met (or becomes false)

Loops are used to (but not limited to) reduce the redundancy and repetitiveness of the code.

 

Exiting a loop

There are several ways to exit a loop.
The normal method is to allow the loop to continue until the loop condition becomes false.

// sentinel / flag

 

Otherwise, we can also force the loop to exit, rather than wait until the code block in the loop ends.

We can do this using the  keyword.

 

 

 loops

the  loop is the basic type of loop that keeps repeating a code until a condition is false.

Example

#include <stdio.h>

int main() {

int num = 1;

while (num < 11) {

printf("The current number is %d.\n", num);

num = num + 1 // We can also write num++

}

return 0;

}

The current number is 2.

The current number is 3.

The current number is 4.

The current number is 5.

The current number is 6.

The current number is 7.

The current number is 8.

The current number is 9.

The current number is 10.

 

REMEMBER

Don't forget to somehow modify the variables in the condition, otherwise it might enter an infinite loop!
