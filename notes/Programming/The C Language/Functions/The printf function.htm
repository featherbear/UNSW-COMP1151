String formatting

The allows you to "format" your string by using placeholders to represent data that can later be added in.

 

For example

#include <stdio.h>

int main(void) {

  printf("Hello my name is %s, I go to UNSW", "Andrew");

  return 0;

}

 

Result

Your output would be "Hello my name is Andrew, I go to UNSW"

What is happening here, is that the string "Andrew" is being substituted into the place where the %s is.
