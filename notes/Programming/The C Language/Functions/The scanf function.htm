The  function allows you to interact with your program, being able to type use the input you provide.

 

Getting started

The function reads the user's input of the program into a variable.

 belongs to the  library, so we must load the library into our program:

 

For example

#include <stdio.h>

int main(void) {

  str name;

  scanf("What is your name? ", &name);

  printf("Hey there, %s!", name)

  return 0;

}

 

Result

> What is your name? Andrew

Hey there, Andrew!

 

IMPORTANT!

See the ampersand in the line scanf("What is your name? ", &name)?

This is an important character to add, or else the program will not work!

In the C language, a variable prepended by an ampersand (ie &var) makes the variable a pointer.

For simplicity, this tells the  function into the existing variable , rather than a new variable.

 

scanf return values

scanf returns the number of values that is has successfully matched

 

Summary

  1. #include <stdio.h>
  2. Declare a variable
  3. Use the  function to read the user input into the variable.
  4. Put an ampersand before the name of the variable!
