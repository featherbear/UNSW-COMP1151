The  command displays a single given character.

 

Getting started

In your code, at the top of the file, enter the line:

Note: Characters must be enclosed with single quotes (')

 

Function Declaration

int putchar(int char);

 

Return Value

This function returns the character written as an unsigned char cast to an int or EOF on error.

 

Usage

char c = 'A'; // We must use the single quote ('), not a double quote (")

putchar(c);
