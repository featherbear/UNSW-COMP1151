When a computer runs a C program, it needs to know where to start the program.
In C, this entry point is the function called .

 

program.c

#include <stdio.h>

int main (int argc, char *argv[]) {

    printf("%d\n", argc);

 

    int i = 0;

    while (i < argc) {

        printf("Argument %d is: %s\n", i, argv[i]);

        i++;

    }

}

 

gcc program.c -o program
