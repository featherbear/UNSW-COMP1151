The  command prints out whatever it is given into a file of your choice.

The same string formatting rules from  apply

 

Syntax

The same as the  function, but we can specify the output file/stream with the first parameter

 

For example

// These two lines do the exact same thing

fprintf(stdout, "Hello, World!");

printf("Hello, World!");

 

// We can print into a file too

FILE* file = fopen("hello.txt", "w");

fprintf(file, "Hello, World!");
