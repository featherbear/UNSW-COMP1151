Misleading Function Names

Wednesday, 25 April 2018

1:39 PM

int isEven(int number) {

    return number % 2;
}

 

What is the return value of isEven(5)

 

1- True (1)

2- False (0)

 

Answer

True!

5mod2 is 1.

Make sure that when you write functions, you name them appropriately
