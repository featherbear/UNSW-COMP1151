Which expression can be used to replace the expression in:

if (!A && !B) { ... }

 

1- !(A && B)

2- !(A || B)

3- !A + !B

4- !(!A + !B)

 

 

Answer

!(A || B)

 

 

 

Other expressions

From: !A && !B

 

= Incorrect =

!(A && B)

!A + !B

!(!A + !B)

!(A * B)

A || B

 

= Correct =

!(A + B)

!A * !B

 

Proof (JavaScript)

[

 "A=true; B=true;",

 "A=false; B=false;",

 "A=true; B=false;",

 "A=false; B=true;"

].map(vars => {

    eval(vars);

    console.log([

                 "!(A && B)",

                 "!(A || B)",

                 "!A + !B",

                 "!(!A + !B)",

                ].filter(a =>  !(eval(a)) == !(!A && !B)  )

               );

});
