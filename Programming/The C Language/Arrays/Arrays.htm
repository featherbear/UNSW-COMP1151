What are arrays?

Think of an  as a list of the same type of items.

A list of numbers, a list of prices, etc…

 

We generally use arrays when we want to group a collection of data items together.

 

One could store each item like

int item1 = 1;

int item2 = 2;

int item3 = 3;

int item4 = 4;

int item5 = 5;

 

…

But, it won't work very well if we had 1000 items to store, let alone it being time consuming.

 

Instead, arrays can be used to efficiently carry out this task

int item[3]; // The 5 represents the size of the array (how many things are we storing)

item[0] = 1; // Item of index 0 (first item)

item[1] = 2; // Index one (second item)

item[2] = 3; // Index two (third item)
